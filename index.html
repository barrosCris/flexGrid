Sistema de grids flex CSS/SASS

Introdução

Esse sistema de grids foi criado utilizando o método flex-box. Ele incorpora uma abordagem mais simplificada e de melhor renderização no browser.
Em conjunto com SASS, esse projeto permite uma configuração personalizada bem facilitada.

Quick start

Para inserir este componente em su projeto, basta copiar o arquivo /partials/_grid.scss para o seu projeto.

Configurações

Para configurar os tamanhos de breakpoints, altere as variáveis o arquivo /variables/_breakpoinrts.scss.
Para configurar nomes de variável, espaçamentos e quantidade de colunas, altere as variáveis o arquivo /variables/_grid-config.scss.

Utilização

Para utilizar a grid, crie uma div com a classe .grid, e para cada coluna utilize a classe .col-{nº de colunas}.
Caso queira centralizar os itens da grid utilize a classe .center-items junto com a classe .grid.
Caso queira criar breakpoints utilize as classes .col-tablet-{nº de colunas} e .col-mobile-{nº de colunas}. 

                
                    <div class="grid">
                        <div class="col-3 col-tablet-6 col-mobile-12">3</div>
                        <div class="col-3 col-tablet-6 col-mobile-12">3</div>
                        <div class="col-3 col-tablet-6 col-mobile-12">3</div>
                        <div class="col-3 col-tablet-6 col-mobile-12">3</div>
                    </div>
